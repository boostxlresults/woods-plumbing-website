# Wood's Plumbing - Next.js Implementation Guide
## Project Context
This is a professional plumbing services website for Wood's Plumbing Enterprises LLC, migrated from React+Vite (CSR) to Next.js (SSR) for proper SEO optimization. The site features 66 services, 8 locations, 22 blog posts, and comprehensive SEO including AI search engine optimization.
## Phase 1: Project Setup
1. **Initialize Next.js with TypeScript**
   - Use App Router (not Pages Router)
   - Enable TypeScript strict mode
   - Configure src/ directory structure
2. **Install Core Dependencies**
   ```bash
   npm install @neondatabase/serverless drizzle-orm drizzle-zod
   npm install @radix-ui/react-* shadcn/ui components
   npm install tailwindcss @tailwindcss/typography
   npm install zod react-hook-form @hookform/resolvers
   npm install date-fns class-variance-authority clsx tailwind-merge
   npm install lucide-react react-icons
   npm install -D drizzle-kit @types/node

Configure Environment Variables
DATABASE_URL=<from Replit secrets>
NEXT_PUBLIC_SITE_URL=https://woodsplumbingaz.com

Phase 2: Database Setup
Copy Schema - Import the provided lib/db/schema.ts
Configure Drizzle - Set up drizzle.config.ts for Neon PostgreSQL
Database Connection - Create lib/db/index.ts with Neon serverless client
Seed Database - Create seed script to populate all tables from JSON files:
66 services from services.json
8 locations from locations.json
38 FAQs from faqs.json
10 reviews from reviews.json
22 blog posts from blog-posts.json
Phase 3: Core Setup
Business Constants - Import lib/constants.ts (company info, hours, social, etc.)
SEO Schemas - Import lib/seo/schemas.ts (Schema.org generators)
Tailwind Config - Set up with shadcn/ui theme tokens
Layout Component - Create root layout with:
Header with logo, navigation, phone CTA
Footer with business info, hours, links, certifications
Mobile floating action buttons (Call Now / Book Online)
