# Next.js Starter Project

## Overview
This is a Next.js application bootstrapped using `create-next-app`. It provides a foundation for building web applications with Next.js, TypeScript, and Tailwind CSS.

## Current State
- **Framework**: Next.js 15.2.3
- **Runtime**: Node.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.0.15
- **Dev Server**: Running on port 5000

## Project Structure
- `pages/` - Next.js pages and API routes
  - `index.tsx` - Main landing page
  - `api/hello.ts` - Example API route
- `styles/` - Global and module CSS files
- `public/` - Static assets

## Recent Changes
- **2025-11-19**: Configured dev server to run on port 5000 for Replit compatibility

## Getting Started
Run `npm run dev` to start the development server. The app will be available at your Replit URL.
